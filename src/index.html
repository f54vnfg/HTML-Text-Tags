HTML Tags

<b> and <strong>

Both of these tags will make text look bold. The <strong> tag also describes higher text importance than the first. Also, according to the HTML 5 specification, the <b> tag should only be used as a LAST resort, when no other tag is more appropriate.

HTML:

<b>bold text</b>

Result:

bold text

<i> and <em>

Both tags will make text look italic. The <em> tag also describes a higher emphasis than the first. Also, according to the HTML 5 specification, the <i> tag should be only used as a LAST resort when no other tag is more appropriate.

HTML:

<i>italic text</i>

Result:

italic text

<span>

This tag is used to group inline elements on the page. The tag itself provides no visual change, but you can use it as a hook to gain access to it with CSS or JS. For example if you want to change a part of a text with different style or layout, you will wrap the desired text in this tag, add some class and target it with CSS.

HTML:

<span>sample text</span>

Result:

sample text

<em>

This is tag used to emphasize parts of a text. Makes a text, or text within a text to stand out a bit. You can also use it as a hook for CSS and JS, as the span tag, but have it be more semantic for particular cases.

HTML:

<em>sample text</em>

Result:

sample text

<code>

This tag is used to define a piece of computer code. You can use it to wrap some of your favorite snippet that you want to post on your site. Like a lot of other tags, it's mostly for writing semantic code. But you can target this particular tag with JS and CSS to make it highlighted like a proper editor and make it look really nice and pretty.

HTML:

<code>sample text</code>

Result:

sample text

<samp>

This tag is used to wrap the output of the computer program. So, for example, if you have a little console web application within your site, you can use this tag to wrap your console command returns. This will make it semantic, and you can use it as a hook to style it or to access it with JS.

HTML:

<samp>sample text</samp>

Result:

sample text

<kbd>

This tag is used to define keyboard input. For example, if you have a little application that requires you to enter something, and there isn't another more semantic tag, you can use this one. Like all others, it can be used as a hook for CSS styling or JS logic.

HTML:

<kbd>sample text</kbd>

Result:

sample text

<var>

This tag is used to define a variable. You might use it as a hook and style specific piece of text which is your variables within a text or stands alone text.

HTML:

<var>sample text</var>

Result:

sample text

<br>

This tag inserts a single line break. This line serves as the 'enter' key for the webpage, like in a word document. Also, this is an empty self-closing tag which means that it has no need for a closing tag.

HTML:

<br /> Text

Result:


Text

<hr>

This element is used to separate content or define a change in the HTML page. This tag puts a full line across the whole page within the parent box.

HTML:

<hr /> Text

Result:

Text

<p>

This is the paragraph tag, widely used for containing a block of text. This paragraph tags come some default left margin from the user agent CSS.

HTML:

<p>Text</p>

Result:

Text

<ins>

This tag is to be used to notify users that some new text has been inserted. By default, the browsers display an underline to make the text stand out. As far as I have seen, this reason this the tag is often not seen compared to its counterpart <del>.

HTML:

<ins>Text<ins>

Result:

Text

<s>

This tag is similar to the <del> tag, The difference is that text is not relevant anymore, yet not it's not deleted. This tag is usually used by online shops to highlight a price discount.

HTML:

Price: <s>£199</s>

Result:

Price: £199

<del>

This tag is to be used on text that is deleted but still kept as a reference. This tag is usually used in blogs websites where the incorrect text is marked, but not deleted.

HTML:

It was my <del>worst</del> idea.

Result:

It was my worst best idea.

<mark>

This tag is to be used to define and highlight parts of your text in a way a marker would on a piece of paper.

HTML:

<mark>Text</mark>

Result:

Text

<sup>

This tag defines superscript text. Superscript text appears half a character above the normal line and is rendered in a smaller font.

HTML:

5<sup>th<sup/>

Result:

5th

<sub>

This tag defines subscript text. Subscript text appears half a character below the normal line and is rendered in a smaller font.

HTML:

H <sub>2<sub/>O

Result:

H2O

<cite>

This tag is used to define text that you would want to reference. In HTML5 this should refer to the text itself and not the author.

HTML:

<cite>Text</cite>

Result:

Text

<abbr>

This is the new acronym tag and it's used for abbreviations. Basically, if you have a shortened term like 'WHO' you can have this term in <abbr> tag, and when the user hovers over the term he will get a pop-up with full text.

HTML:

<abbr class="World Health Organization">Text</abbr>

Result:

WHO

<dfn>

This tag is used to explain new terminologies. They call it the defining instance of a new term. So if you are describing something complicated like a hedge fund you would use this tag. This will be used by screen readers, most browsers would not do anything to the text itself, but some would make the text italic.

HTML:

<dfn title="Text">Text</dfn>

Result:

Text

<q>

This tag is used for short quotation and the browser usually insert quotation marks around the quote.

HTML:

<q>Text</q>

Result:

Don't cry because it's over, smile because it happened. – Dr. Seuss

<small>

This tag is used to define small text or any side comment that you want to deemphasize. It is usually used within the footer of the site for the copyright info as by default as it's smaller.

HTML:

<small>small text</small>

Result:

small text

<backquote>

This tag is used to define and reference a full paragraph quote. This tag is very similar to the <q> tag, but the <q> is for short one or two sentences quotes.

HTML:

<backquote>

<p>

You've gotta dance like there's nobody <br>
watching, Love like you'll never be hurt, Sing like <br>
there's nobody listening, And live like it's heaven <br>
on earth. – William W. Purkey

</p>

</backquote>

Result:

You've gotta dance like there's nobody
watching, Love like you'll never be hurt, Sing like
there's nobody listening, And live like it's heaven
on earth. – William W. Purkey

<address>

This is the tag used for addresses, it's pretty much self-explanatory. just use it in combination with other tags like </br>, </div> with a indent margin class to properly display your address.

HTML:

<address>

Business Name <br>
The Parkside Building <br>
5 Cardigan St <br>
Birmingham <br>
B4 7BD <br>

<address>

Result:

Business Name
The Parkside Building
5 Cardigan St
Birmingham
B4 7BD
