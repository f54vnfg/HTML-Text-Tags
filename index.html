HTML BASIC TAGS

<b> and <strong>

Both tags would make a text look bold. However, the second tag also describes a higher importance of the text. Also, according to the HTML 5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate.

HTML : <b>bold text</b>.

Result : bold text.

<i> and <em>

Both tags would make a text look italic. However, the second second tag also describes a higher emphasis of the text. Also, according to the HTML 5 specification, the <i> tag should be used as a LAST resort when no other tag is more appropriate.

HTML: <i>italic text</i>

Result: italic text.

<br>

This tag inserts a single line break. In simple terms, this line serves as the 'enter' key for the webpage as it does on a word document. Also, this is an empty self-closing tag which means that it has no need for a closing tag.

Example : <br /> Text

Result:
Text

<hr>

This element is used to separate content (or define a change) in an HTML page. In simple terms, this tag puts a nice line across the whole page within the box model of the parent element, and it's mainly used as a separator for different sections.

Example : <hr /> Text

Results:

Text

<p>

This is the paragraph tag, widely used for containing a block of text. Also, this paragraph has some left margin by default from the user agent CSS.

HTML: <p>Text</p>

Result:

Text

<ins>

This tag is mean to be used on some text that has been inserted into the text. This is usually used in blogs to showcase updated text. However, by having an underline it makes the text stand out, as far as I have seen this reason this the tag is often left out.

HTML: <ins>Text<ins>

Result: Text

<s>

This tag is similar to the <del> tag, but here the text is meant to be not relevant anymore, not deleted. Also, this tag can be used in online shops to highlight the price discount by having the normal price with a line through it.

HTML: Price: <s>£199</s>

Result: Price: £199

<del>

This tag is meant to use on some text that has been deleted but is kept as a reference. This tag is usually used in blogs where incorrect is marked but not deleted. Usually to show that the text was wrong but was corrected. These tags are givng meanings to the screen readers, both tags display idetically in the browser.

HTML: It was my <del>worst</del> idea.

Result: It was my worst best idea.

<mark>

This tag is mean to be used to define and highlight parts of your text.

HTML: <mark>Text</mark>

Result: Text

<sup>

This tag defines superscript text. Superscript text appears half a character above the normal line and is sometimes rendered in a smaller font.

HTML: 5<sup>th<sup/>

Result: 5th

<sub>

This tag defines subscript text. Subscript text appears half a character below the normal line and is sometimes rendered in a smaller font.

HTML: H <sub>2<sub/>O

Result: H2O

<cite>

This tag is used to define text that you would like to reference. In HTML5 this should refer to the text it self and not the author.

HTML: <cite>Text</cite>

Result: Text

<abbr>

This is the new acronym tag and it's used for abbreveiation of text. Bascially if you have a shorten term like 'NSA' you can have this text around this tag and when user hovers over it he will get the full term. This is also used by screen readers.

HTML: <abbr>Text</abbr>

Result: Text

<dfn>

This tag is used to explain new terminolgoies. They call it as the defining instance of a new term. So if you have explain something like for example a hedge fund you would use this tag around that word. This will be used by screen readers, most browsers would not do anything to the text it self, some would make the text italic.

HTML: <dfn>Text</dfn>

Result: Text

<q>

This tag is used for short questions and browser usually insert quotation marks around the quote.

HTML: <q>Text</q>

Result: Don't cry because it's over, smile because it happened. – Dr. Seuss

<backquote>

This tag is used to define and reference a full paragraph quote. This tag is very similar to the <q> tag but where the <q> is for short quotes, this tag is for full paragraph quote.

HTML:

<backquote>
<p>
You've gotta dance like there's nobody <br>
watching, Love like you'll never be hurt, Sing like <br>
there's nobody listening, And live like it's heaven <br>
on earth. – William W. Purkey
</p>
</backquote>

Result:


You've gotta dance like there's nobody
watching, Love like you'll never be hurt, Sing like
there's nobody listening, And live like it's heaven
on earth. – William W. Purkey

<address>

This is the tag used for addresses, it's pretty much self-explanatory. just use other tags like </br>, </div> or </p> with a indent class to properly display your address.

HTML:


<address>
Business Name <br>
The Parkside Building <br>
5 Cardigan St <br>
Birmingham <br>
B4 7BD <br>
<address>

Result:


Business Name
The Parkside Building
5 Cardigan St
Birmingham
B4 7BD
