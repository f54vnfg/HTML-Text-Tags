HTML5: The Basics Tags

<b> and <strong> Both tags would make a text look bold. However, second tag also describes a higher importance of the text. Also, according to the HTML 5 specification, the <b> tag should be used as a LAST resort when no other tag is more appropriate.

HTML : <b>bold text</b>.

Result : bold text.

<i> and <em> Both tags would make a text look italic. However, second tag also describes a higher emphases of the text. Also, according to the HTML 5 specification, the <i> tag should be used as a LAST resort when no other tag is more appropriate.

HTML: <i>italic text</i>

Result: italic text.

<br> This tag inserts a single line break. In simple terms, this line serves as the 'enter' key for the webpage as it does on a word document. Also, this is an empty self closing tag which means that it has no need for a closing tag.

Example : Text <br />

Result:
Text

<hr> This element is used to separate content (or define a change) in an HTML page. In simple terms this tag puts a nice line accross the whole page within the box model of the parent element, and it's manly used as a separator for different sections.

Example : Text <hr/>

Results: Text

<p> This is the paragraph tag, widley used for containing a blocks of text. Also, this paragraph has some left margin by default from the user agent CSS.

HTML: <p>Text</p>

Result:

<q> This tag is used for short quatitons and browser usually insert quatation makrs around the quote.

HTML: <q>Text</q>

Result: Text

<ins> This tag is mean to be used on some text that has been inserted into the text. This is usally used in blogs to showcase updated text. However, with having an underline it makes the text stand out, so for this reason this tag is often left out.

HTML: <ins>Text<ins>

Result: Text

<del> This tag is meant to used on some text that has been deleted, but is kept as a reference.This tag is usally used in blogs where incorrect is marked but not deleted. Ususally to show that the text was wrong but was corrected. Also, this tag can be used in online shops to highlight price discount.

HTML: It was my <del>worst</del> idea.

Result: It was my worst best idea.

<s> This tag is similar to the <del> tag, but here the text is mean to be irrelevent, not deleted. These are meaning givng to the screen readers, both tags display idetically in the browser.

HTML: Price: <s>£199</s>

Result: Price: £199

<mark> This tag is mean to be used to define and highlight parts of your text.

HTML: <mark>Text</mark>

<sup> This tag defines superscript text. Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font.

HTML: 5<sup>th<sup/>

Result: 5th

<sub> This tag defines subscript text. Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font.
